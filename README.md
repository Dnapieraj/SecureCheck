# SecureCheck — URL & File Scanner (VirusTotal)

Modern, lightweight web app to scan URLs and files using the VirusTotal API. Dark theme, animated UI, stacked progress bars, and a detailed per-engine report.

## Features
- Scan URLs and files (drag-and-drop style upload area)
- Aggregate result: malicious / suspicious / harmless / undetected
- Animated stacked progress bar with percentages
- Detailed report modal (engine → result)
- Responsive, dark, glassmorphism UI

  
## Requirements
- VirusTotal API key (personal)

## Download & Installation
